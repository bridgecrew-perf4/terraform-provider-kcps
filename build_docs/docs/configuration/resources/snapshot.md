# Snapshot

## 使用例

```hcl
resource kcps_snapshot "my_snapshot"{
    volumeid = "453983a0-5c2e-4645-9468-5e7e63ebf045"
}
```


## パラメータ

|パラメータ名 |必須    |説明      |初期値    |タイプ    |備考|
|:----------|:------|:---------|:--------|:--------|:--|
|`volumeid` |◯|スナップショット作成に使用するボリュームのID  | - | string | - |


## 属性
|属性名 |説明      |備考 |
|:----------|:------|:---------|
|`id`          |リソースID   | - |
