# Zone

## 使用例

```hcl
data kcps_zone "my_zone"{
    name = "jp2-east03"
}
```

## パラメータ

|パラメータ名 |必須    |説明      |初期値    |タイプ    |補足|
|:----------|:------|:---------|:--------|:--------|:--|
|`zone_id` |-|ZoneのID | - | string | - |
|`name` |-|Zoneの名前 | - | string | APIにはない機能 |


## 属性

|属性名 |説明      |補足 |
|:----------|:------|:---------|
|`id`      |データソースID   | - | 
|`zone_id`  |ZoneのID  | `id`と同じ値 |
|`name` |Zoneの名前 | - |
